# 106-2-R-project
## 以R語言分析台灣觀光產業
### 資料來源:政府開放資料平台、高雄市政府

1.熱力圖分析: <br>
  利用cor()函數了解「年份」、「人數」和「收入」三者之間的相關性，並繪製成熱力圖以方便知悉三者間的關聯。由熱力圖上的顯示結果可以得知人數與收入之間呈現低度相關，亦即人數並非決定收入的主要因素。
